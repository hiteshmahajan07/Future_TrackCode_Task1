# 🛒 Business Sales Dashboard | Power BI

## 📌 Project Overview
This project focuses on analyzing **e-commerce (supermarket) sales data** to uncover key business insights such as best-selling products, revenue trends, and category performance.

An interactive dashboard is built using **Power BI** to support data-driven decision-making.

---

## 🎯 Objectives
- Identify **top-performing products**
- Analyze **monthly and daily sales trends**
- Evaluate **profitability and revenue**
- Understand **customer payment behavior**
- Compare performance across **product categories**

---

## 📊 Dashboard Highlights
- 💰 **Total Sales:** 201K+
- 📈 **Total Profit:** 68K+
- 📊 **Profit %:** 21%
- 📅 Monthly Sales Trend Analysis
- 🛍️ Sales by Product & Category
- 💳 Payment Mode Distribution
- 📆 Daily Sales Insights

---

## 🧰 Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation

---

## 📂 Dataset
The dataset contains:
- Sales transactions
- Product categories
- Payment modes
- Date-wise sales records

---

## 📈 Key Insights
- Certain categories contribute significantly to total revenue
- Sales show variation across months indicating seasonal trends
- Online payment methods are widely used
- Profit margins remain stable across categories
